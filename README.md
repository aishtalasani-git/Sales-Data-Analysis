# Sales Data Analysis using Python

## Project Overview
This project analyzes retail sales data using Python to discover business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Sales by Category
- Profit by Region
- Sales by Segment
- Top 10 States by Sales
- Monthly Sales Trend
- Top 10 Products by Sales
- Discount vs Profit Analysis
- Correlation Analysis
- Dashboard

## Key Insights
- Technology generated the highest sales.
- West region recorded the highest profit.
- Consumer segment generated the highest sales.
- California achieved the highest sales among all states.
- Higher discounts generally reduced profit.

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insight Generation
